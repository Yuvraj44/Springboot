"/auth/**" -> permitted

"/auth/register" 
If we register, check if the user already exists.
If user doesn't exist
1. encrypt the password 
2. Save the user with encrypted password
3. Generate a token for the user
4. Map the token to the User


"/auth/confirm"
If we want to confirm, we send a token:
1. Check if token is in DB
2. Check if token is not confirmed yet.
3. Check that token is not expired.
4. Enable the user.
